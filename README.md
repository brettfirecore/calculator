# Format Specifiers
    %d:   Integer
    %s:    String
    %f:     Floating-point number
    %.nf:  Floating-point number with n decimal places
         (%.1f  "tenths") %.2f  %.3f
    %t:     Boolean
    %v:     Default format for the type
    %T:    Type of the value
